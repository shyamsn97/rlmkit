FILES:
- index.html: HTML entry. Include <canvas id="canvas"></canvas>, link style.css, include scripts in order: utils.js, boid.js, flock.js, main.js. Include overlay elements for FPS and count.
- style.css: Styles for body, canvas (fullscreen), overlay.
- utils.js: Plain JS helpers: vec operations (create, add, sub, mult, div, mag, setMag, limit), wrapPosition.
- boid.js: class Boid with update/applyBehaviors/edges/draw. Colorful hues per boid.
- flock.js: class Flock managing array of Boid, updating and drawing.
- main.js: Setup canvas, create Flock with ~300 boids, animation loop, FPS & count overlay, window resize handler.

Write the single target file path as output/boids-simulation/<filename>.
